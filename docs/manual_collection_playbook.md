# 广西阳光志愿系统手工取数流程

本文档用于指导在已经登录广西阳光志愿信息服务系统后，如何手工获取本项目需要的数据。

## 一、先明确什么最值得拿

优先级按下面顺序来：

1. `招生计划`
2. `选院校`
3. `投档分数明细`
4. `分数线`
5. `一位一档 / 历年一位一档`

原因：

- `招生计划` 最适合拿 2026 年正式计划、专业组、组内专业、人数、学费、学制、备注。
- `选院校` 最适合拿历年院校专业组投档情况和筛选候选池。
- `投档分数明细` 最适合补专业层面的往年分布。
- `一位一档` 更像查询工具，不一定适合一次性导出全表。

## 二、登录后先做什么

PC 端登录后，先不要急着搜学校。

先做三件事：

1. 打开 `我的成绩`
2. 录入你的科目类型和成绩
3. 确认系统已经识别为 `物理类`

这样后面的院校、专业、计划会更贴近你的科目口径。

## 三、怎么拿 2026 招生计划

这是最重要的一步。

点击顶部菜单里的 `招生计划`，优先按下面条件筛：

- 招生年份：`2026`
- 录取批次：`本科普通批`
- 科类：`物理类`
- 院校所在省份：先不选，或者按省份分批查

然后这样操作：

1. 先不填院校名，直接查全量结果。
2. 如果结果太多，就按省份分开查。
3. 每查出一页，就把页面另存为 `完整网页` 或打印成 `PDF`。
4. 文件名统一成：

```text
raw_data/2026/plan_2026_physics_benke_<省份或all>_<页码>.html
```

页面里重点保留这些字段：

- 院校代码
- 院校名称
- 专业组代码
- 专业名称
- 计划数
- 学费
- 学制
- 备注
- 选科要求

如果系统允许展开专业组详情，就把展开后的页面也保存下来。

## 四、怎么拿历年投档情况

点击 `选院校`。

推荐这样筛：

- 招生年度：先查 `2025`，再查 `2024`
- 录取批次：`本科普通批`
- 科目组：`物理类`

然后按两种方式拿：

### 方式 A：按学校查

适合做目标院校池。

1. 输入院校名，例如某 211、双一流、强双非。
2. 进入院校详情。
3. 查看该校在广西的专业组及历年投档信息。
4. 如有 `投档分数明细`，点进去。
5. 保存页面。

文件名建议：

```text
raw_data/2025/school_2025_<院校代码或院校名>.html
raw_data/2024/school_2024_<院校代码或院校名>.html
```

### 方式 B：按筛选条件扫候选池

适合快速找“能报的组”。

可按这些条件筛：

- 院校层次
- 所在省份
- 是否公办
- 专业关键词

适合先搜：

- 计算机
- 软件工程
- 人工智能
- 电子信息
- 自动化
- 通信
- 数学
- 物理

## 五、怎么拿专业层面信息

点击 `选专业`。

这是补充，不是第一优先级。

适合做两件事：

1. 搜目标专业，反查哪些学校/专业组包含它。
2. 搜不想要的专业，确认哪些组含禁忌专业。

建议重点搜：

- 计算机类
- 软件工程
- 人工智能
- 电子信息类
- 自动化
- 通信工程

也建议排查这些你可能不接受的组：

- 护理
- 农林
- 矿业
- 地质
- 土木
- 材料
- 化工
- 环境
- 生物
- 食品
- 海洋

## 六、怎么用一位一档

系统里的 `一位一档` / `历年一位一档` 更适合查具体分数对应位次，不一定适合导出完整表。

你可以这样用：

1. 选择年份。
2. 选择 `物理类`。
3. 输入总分。
4. 如果页面要求，再输入同分排序相关字段：
   总分、语数之和、语文或数学单科最高分、外语分、首选科目分、再选科目最高分、再选科目次高分。

它最适合两个场景：

- 查你自己的准确位次
- 把某个投档分反查成位次

它不一定适合整表采集，所以整表仍优先用广西招生考试院官方一分一档页面。

## 七、建议你怎样保存文件

每次保存都按固定规则命名，不然后面会乱。

建议格式：

```text
raw_data/<year>/<type>_<year>_<batch>_<subject>_<keyword>.html
raw_data/<year>/<type>_<year>_<batch>_<subject>_<keyword>.pdf
```

例子：

```text
raw_data/2026/plan_2026_benke_physics_all_p1.html
raw_data/2025/admission_2025_benke_physics_xxx大学.html
raw_data/2024/rank_2024_physics_score590.pdf
```

## 八、你最该先点哪几个地方

如果时间有限，只做这 4 组：

1. `招生计划`
   目标：拿 2026 本科普通批物理类的专业组和专业明细。

2. `选院校`
   目标：拿 2025、2024 的院校专业组投档情况。

3. `投档分数明细`
   目标：补专业层面的往年分布。

4. `我的成绩`
   目标：确认系统按你的口径匹配。

## 九、我们怎么配合最省事

最省事的方式不是你手抄，而是：

1. 你登录系统。
2. 你按上面的入口把关键页面另存为 HTML 或 PDF。
3. 放进 `raw_data/2024/`、`raw_data/2025/`、`raw_data/2026/`。
4. 我来继续做清洗、抽表、转 CSV、建模。

## 十、当前判断

这个系统有用，而且对我们最有价值的是：

- `招生计划`
- `选院校`
- `投档分数明细`

`一位一档` 更适合查单个分数或单个考生位次，不是最适合批量建库的入口。
