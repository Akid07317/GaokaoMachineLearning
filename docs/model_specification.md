# 模型说明

本文档记录项目模型公式、适用条件和回测要求。当前为基础骨架。

## 建模目标

对每个广西物理类本科普通批院校专业组预测：

- 最低投档位次区间：`Q10`、`Q50`、`Q90`
- 考生达到投档线的概率区间
- 风险档：冲、稳中有险、稳、保、垫底保底、不建议、数据不足
- 风险解释

## 基线模型

### 去年位次延续

```text
R_hat_base(g, 2026) = R(g, 2025)
```

适用于专业组结构稳定、计划变化不大的专业组。所有复杂模型必须与此基线比较。

### 趋势收缩

```text
delta_log_R(g, 2025) = log(R(g, 2025)) - log(R(g, 2024))
log_R_hat(g, 2026) = log(R(g, 2025)) + lambda * delta_log_R(g, 2025)
```

`lambda` 应通过回测估计。2026 填报前可先取保守范围 `0.2-0.5` 做敏感性分析。

### 计划修正

```text
log_R_hat(g, y) =
alpha + beta * log(R(g, y-1)) + gamma * log(P(g, y) / P(g, y-1)) + error
```

`gamma` 不能手填，应通过 2024 预测 2025 的回测估计。

## 区间与概率

设专业组最低投档位次随机变量为 `T_g`，考生位次为 `r_c`：

```text
P(admit) = P(T_g >= r_c)
```

概率只输出区间，不输出伪精确小数。

## 回测指标

- `MAE`
- `RMSE`
- `MAPE`
- 区间覆盖率
- 保底失效率
- 冲刺命中率
- 重大误判案例复盘

## 模型禁区

不得使用以下泄漏特征参与填报前预测：

- 2026 投档最低分
- 2026 投档最低位次
- 2026 征集志愿计划
- 2026 录取结果
- 2026 专业录取最低分
- 志愿填报后第三方平台实际填报热度
