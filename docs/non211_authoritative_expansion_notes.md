# 非211扩池说明

这份清单服务于当前 `211 工科目标池` 之外的后续扩池工作，不和现有 `reports/engineering_probe_backlog.csv` 混用。

## 当前筛选口径

第一批只纳入同时满足以下条件的学校：

1. **非211**
2. 在**教育部第二轮“双一流”建设高校名单**内
3. 与当前项目的理工/工科方向有明显相关性，或可作为后排补充

## 权威来源

- “211工程”学校名单（教育部）  
  <https://www.moe.gov.cn/srcsite/A22/s7065/200512/t20051223_82762.html>
- 教育部、财政部、国家发展改革委关于公布第二轮“双一流”建设高校及建设学科名单的通知  
  <https://www.moe.gov.cn/srcsite/A22/s7065/202202/t20220211_598710.html>

## 使用方式

- `core`：后续优先补抓官方本科招生网、招生计划、历年分数
- `support`：作为普通批后排扩池候选
- `special_review`：招生机制可能与普通批不完全同口径，先核查再决定是否并入主池

## 当前策略说明

这份清单是**保守起步版**。我们先用教育部口径把明显值得追的非211强校挂进待办，再决定是否扩展到：

- 省属高水平大学
- 行业特色强校
- 非双一流但信息/电子/自动化方向很强的学校

这样能先保证扩池来源干净，再逐步放宽。
